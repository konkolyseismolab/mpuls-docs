Template Class CIRFlux

  • Defined in File flux_operators.hpp

Class Documentation

template<QuantityType Q, Direction Dir, EquationType<Q> Eq>
class CIRFlux

Local CIR flux.

It is by definition:

\[\begin{split} F_{i+1/2} = \left\lbrace \begin{array}{lr} F_{i+1/2}^+ & \text{if} v < 0 \\ F_{i+1/2}^- & \text{if} v > 0 \\ 0 & \text{elsewhere} \end{array} \right. \end{split}\]

Where \(\lambda\) is the largest characteristic speed at the interface i.e. \( \max [ F'_+ , F'_-]]\), where F’ = (\partial F / \partial q) eigenvalues. And +,- means the two sides of the interface.
