Class Documentation

template<typename D, int N>
class QuantityBase

The base class that can be used to define simple state-vectors.

Basic operations (+,-,*,/) are overloaded for clarity. Mathematically this is a templated implementation of e.g.

\[\begin{split} \boldsymbol q = \left( \begin{array}{c} \rho \\ u \\ v \\ p \end{array} \right) \end{split}\]

Note

The actual meaning of the values inside state_vector depends on the user context of the actual object that uses it.

Template Parameters:

  • D – The floating point type

  • N – The size of the state-vector

Prevents zero division. This function gives a small number to the state vector elements, which prevents zerodivision in 1/Q situations.
