template<QuantityType Q, class RHS, class CFL, class Boundary>
class SSP_RK2 : public TimeSolverBase<Q>

Strong Stability Preseving Runge Kutta 2nd order.

This solver implements the SSP-RK2 method, also known as Heun method for ordinary differential equations, in the method of lines sense. The method is described in 2 steps. Giving the following formula:

\[\begin{split} \begin{array}{c} \displaystyle \boldsymbol{k}_1 = h \boldsymbol{f}(\boldsymbol{q}^n) \\ \displaystyle \boldsymbol{k}_2 = h \boldsymbol{f}(\boldsymbol{q}^n + k_1) \\ \displaystyle \boldsymbol{q}^{n+1} = \boldsymbol{q}^n + \frac 1 2 (\boldsymbol k_1 + \boldsymbol k_2) \end{array} \end{split}\]

Template Parameters:

  • Q – The structure of the State vector

  • RHS – The right hand side of the equation \(\boldsymbol{f}(\boldsymbol{q}, t)\)

  • CFL – The Adaptive timestep generator class

  • Boundary – The boundary condition implementation.
