Time integration

Using the method of lines (MOL) aprroach, the semi-difference flux formulas convert the PDE system into a system of ordenary differential equations for each grid points. Therefore the whole system for the \(i,j,k\) gridpoint gives:

\[ \frac{\partial \boldsymbol q_{i,j,k}}{\partial t} = \boldsymbol f (\boldsymbol q_{i,j,k}, \boldsymbol r_{i,j,k}, t) \]
group TimeSolvers

The time solvers have a common virtual interface defined in TimeSolverBase. They implement various Strong Stability Preserving Runge-Kutta schemes (SSP-RK) for temporal integration in the method of lines framework.

The TimeSolverType concept enforces that only concrete implementations (not the base) can be instantiated, preventing improper use of the abstract interface.

template<QuantityType Q>
class TimeSolverBase
#include <time_solver/time_solver.hpp>

The abstract interface for time solvers.

Defines the common virtual interface that all concrete time integration schemes must implement. This includes methods for time stepping control and the main solve routine.

Template Parameters:

Q – Type of the state vector quantity

Subclassed by SSP_RK2< Q, RHS, CFL, Boundary >, SSP_RK3< Q, RHS, CFL, Boundary >

template<QuantityType Q, class RHS, class CFL, class Boundary>
class SSP_RK2 : public TimeSolverBase<Q>
#include <time_solver/time_solver.hpp>

Strong Stability Preseving Runge Kutta 2nd order.

This solver implements the SSP-RK2 method, also known as Heun method for ordinary differential equations, in the method of lines sense. The method is described in 2 steps. Giving the following formula:

\[\begin{split} \begin{array}{c} \displaystyle \boldsymbol{k}_1 = h \boldsymbol{f}(\boldsymbol{q}^n) \\ \displaystyle \boldsymbol{k}_2 = h \boldsymbol{f}(\boldsymbol{q}^n + k_1) \\ \displaystyle \boldsymbol{q}^{n+1} = \boldsymbol{q}^n + \frac 1 2 (\boldsymbol k_1 + \boldsymbol k_2) \end{array} \end{split}\]

Template Parameters:

  • Q – The structure of the State vector

  • RHS – The right hand side of the equation \(\boldsymbol{f}(\boldsymbol{q}, t)\)

  • CFL – The Adaptive timestep generator class

  • Boundary – The boundary condition implementation.

template<QuantityType Q, class RHS, class CFL, class Boundary>
class SSP_RK3 : public TimeSolverBase<Q>
#include <time_solver/time_solver.hpp>

Strong Stability Preseving Runge Kutta 3rd order.

This solver implements the SSP-RK3 method for ordinary differential equations, in the method of lines sense. The method is described in three steps. Giving the following formula:

\[\begin{split} \begin{array}{c} \displaystyle \boldsymbol{k}_1 = h \boldsymbol{f}(\boldsymbol{q}^n) \\ \displaystyle \boldsymbol{k}_2 = h \boldsymbol{f}(\boldsymbol{q}^n + k_1) \\ \displaystyle \boldsymbol{k}_3 = h \boldsymbol{f}(\boldsymbol{q}^n + \frac 1 4 (\boldsymbol k_1 +\boldsymbol k_2 )) \displaystyle \boldsymbol{q}^{n+1} = \boldsymbol{q}^n + \frac 1 3 (\boldsymbol k_1 + \boldsymbol k_2) + \frac 2 3 \boldsymbol k_3 \end{array} \end{split}\]

Template Parameters:

  • Q – The structure of the State vector

  • RHS – The right hand side of the equation \(\boldsymbol{f}(\boldsymbol{q}, t)\)

  • CFL – The Adaptive timestep generator class

  • Boundary – The boundary condition implementation.
