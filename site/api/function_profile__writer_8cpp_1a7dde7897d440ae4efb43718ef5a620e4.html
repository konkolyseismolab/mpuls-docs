void writeProfileToHdf5(const std::string &filename, const std::vector<double> &radius, const std::vector<double> &kappa, const std::vector<double> &xi, const std::vector<double> &l)

Writes a dummy radial profile to an HDF5 file.

This function creates an HDF5 file and stores three 1D datasets:

  • r: radial coordinate (e.g., in cm)

  • kappa: opacity (e.g., in cm²/g)

  • xi: dimensionless auxiliary quantity

All vectors must be of equal length.

Parameters:

  • filename – Name of the HDF5 file to create.

  • radius – Vector of radial coordinates.

  • kappa – Vector of opacity values.

  • xi – Dimensionless radial displacement vector.
