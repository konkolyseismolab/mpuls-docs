Limiters

Limiters are used for solution reconstruction .

template<QuantityType Q>
class LimiterBase
#include <advection/flux_operators.hpp>

The abstract interface for the limiters.

Technically it is just a callable, with two variables.

Subclassed by MinMod< Q >, Superbee< Q >, VanLeer< Q >

template<QuantityType Q>
class VanLeer : public LimiterBase<Q>
#include <advection/flux_operators.hpp>

Van Leer type limiter. It returns the harmonic means of the slopes.

The Van Leer flux limiter limits the forward and backward slope by taking their harmonic mean:

\[\begin{split} \delta q_i = \left\lbrace \begin{array}{ll} \frac{2 \delta q_{i-1/2} }{\delta_q{i+1/2}+\delta{i-1/2}} & \text{if } \delta q_{i+1/2} \delta q_{i-1/2} > 0 \\ 0 & \text{elsewhere} \end{array}\right. \end{split}\]

See also

Reconstruction for how this is used in the hypervolic solver.

Note

Current definition is divided by the LAx-Wendroff slope ($\delta q_{i+1/2}) this may change in the future.

template<QuantityType Q>
class Superbee : public LimiterBase<Q>
#include <advection/flux_operators.hpp>

Superbee type limiter which returns the highest TVD slopes.

This type of limiter allows the highest stable slopes, in the expense of numerical artifacts. The limiter definition is:

\[ \phi(\delta q_{i-1/2}, \delta q_{i+1/2}) = [{\rm sgn} (\delta q_{i-1/2}) + {\rm sgn} (\delta q_{i+1/2})] \min \left[ \vert \delta q_{i-1/2} \vert, \vert \delta q_{i+1/2} \vert, \frac{1}{2} \max (\vert \delta q_{i-1/2} \vert, \vert \delta q_{i+1/2} \vert) \right] \]

Warning

This flux limiter makes slopes into square in some circumstances.

template<QuantityType Q>
class MinMod : public LimiterBase<Q>
#include <advection/flux_operators.hpp>

Minmod type limiter, which returns the smallest second order accurate TVD slope.

The minmod limited slope is:

\[ \phi(\delta q_{i+1/2},\delta q_{i-1/2}) = \min [ \max (\delta q_{i-1/2}, 0) , \max (\delta q_{i+1/2}, 0) ] + \max [ \min (\delta q_{i-1/2}, 0) , \min (\delta q_{i+1/2}, 0) ] \]

template<class T, typename Q>
concept LimiterType
#include <advection/flux_operators.hpp>

C++ concept that forces LimiterBase derived types. The purpose of this to prevent bad substitution into the Reconstruction classes.
