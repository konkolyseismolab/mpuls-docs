inline virtual D getMaxVelocity(const Q &quantity, Direction dir, std::span<const D> coords) const override

Calculates the highest characteristic speed based on the given state-vector.

Some solver needs the largest characteristic speed at the interface (see e.g. LLFflux ). This is e.g. for the Euler system is:

\[ \max \left[ \vert u - c \vert , \vert u \vert , \vert u + c \vert \right] \]

Parameters:

  • quantity – State-vector ( QuantityBase object reference)

  • dir – direction of the interface

  • coords – interface coordinates

Returns:

The maximum characterisctic speed ( \( \max \vert {\rm eig } \partial F / \partial q \vert \) )
