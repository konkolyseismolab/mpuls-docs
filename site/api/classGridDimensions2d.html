inline D interfaceAreaIphalf(size_t i, size_t j)

Calculates interface are at half index vertical coordinate.

The area on the 2D grid (in the current implementation)

\[ A_{i+1/2,j} = 2 \pi r_{i+1/2}^2 \sin \theta_j \Delta \theta \]

Parameters:

i, j – indicies.

Returns:

\(A_{i+1/2,j}\)

inline D interfaceAreaJphalf(size_t i, size_t j)

Computes the are at half-j indices.

At \(j+1/2\) the area is:

\[ A_{i,j+1/2} = 2 \pi r_{i}^2 \sin \theta_{j+1/2} \Delta r_i \]

Parameters:

i, j – indicies.
