Class Documentation

template<QuantityType Q>
class VanLeer : public LimiterBase<Q>

Van Leer type limiter. It returns the harmonic means of the slopes.

The Van Leer flux limiter limits the forward and backward slope by taking their harmonic mean:

\[\begin{split}\delta q_i = \left\lbrace \begin{array}{ll} \frac{2 \delta q_{i-1/2} }{\delta_q{i+1/2}+\delta{i-1/2}} & \text{if } \delta q_{i+1/2} \delta q_{i-1/2} > 0 \\ 0 & \text{elsewhere} \end{array}\right. \end{split}\]

See also

Reconstruction for how this is used in the hypervolic solver.

Note

Current definition is divided by the LAx-Wendroff slope ($\delta q_{i+1/2}) this may change in the future.
