template<QuantityType Q, class FluxOperatorTypeRad, class FluxOperatorTypeHor, class SourceType, EquationType<Q> Equation>
class RightHandSide

The basic two-dimensional semi-difference operator of the right-hand-side of the PDE.

The twod-imensional advection problem in spherical coordinates consists of three operator:

  • vertical advection \(\mathcal F\)

  • horizontal advection @f\mathcal G \( - source term @\mathcal S\)

Note

Template Parameters:

  • Q – a QunatityBase type describing the state-vector

  • FluxOperatorTypeRad – this is the flux operator for the vertical direction

  • FluxOperatorTypeHor – this is the flux operator for the horizontal direction

  • SourceType – this is the source operator of the PDE

  • Equation – the PDE type.
