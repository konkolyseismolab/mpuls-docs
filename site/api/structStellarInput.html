struct StellarInput

Minimal, physics-only input structure for stellar model construction.

The StellarInput structure contains only the physical quantities required to build the stellar model (mass, radius, luminosity, effective temperature, metallicity, and grid configuration). It is intentionally separated from the Config and StellarParameters structures for architectural clarity.
