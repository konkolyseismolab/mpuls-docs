Class Documentation

template<QuantityType Q>
class Grid2d

Templated grid descriptor.

It has N variables and stored as a single continous array. The grid is 2D, with n_radial and n_horizontal zones, and n_radial_ghost and n_horizontal_ghost ghost cells in each direction.

At the moment this is a simple implementation using horizontal rows.

Q – Statevector type of QuantityBase
